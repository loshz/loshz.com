Understanding Mutexes

Oct 19, 2019

Those of us familiar with concurrency and parallelism know just how useful a mutex can be when protecting access to resources. But how do they work?

Here is a basic example in Go of a struct with a count value and a method that increments this value when called:

package main

import "sync"

type protected struct {
	count int
	mtx   sync.Mutex
}

func (p *protected) Inc() {
	p.mtx.Lock()
	p.count++
	p.mtx.Unlock()
}
